Ventajas

Por qué StarTeam

De Excels desconectados a una plataforma unificada que transforma la manera de trabajar.

Trabajo en equipo real

Antes: cada departamento con su Excel, datos duplicados, versiones diferentes, 0 visibilidad cruzada.

Ahora: todos trabajan sobre la misma base de datos, cambios en tiempo real, dashboards compartidos.
Ejemplo: Booking cierra un artista → Producción ve los requisitos → Ticketing abre la venta → Marketing lanza la campaña → Tesorería programa los pagos. Todo conectado.

Base de datos centralizada vs Excels

Una única fuente de verdad (PostgreSQL) con 134 modelos vs 14 Excels desconectados. Sin duplicación de datos, sin errores de copy-paste entre departamentos.

Búsquedas instantáneas, filtros cruzados, exportaciones automáticas. Auditoría completa: quién cambió qué, cuándo, por qué.

Histórico para mediciones reales

548 conciertos históricos (2012-2025) para análisis de rentabilidad real. 20,915 registros F&B (42.89M EUR, 13 ediciones) para benchmarks. 644 cierres de booking para negociación basada en datos.
BCG Matrix, ABC Pareto, CRM enriquecido — decisiones con datos, no con intuición. Simulador booking: "si contratamos a este artista a este caché, ¿cuál es el ROI esperado según histórico?"

Seguridad del software

OWASP Top 10 protegido: SQL injection, XSS, CSRF, path traversal — todo testeado. 5 roles con permisos granulares (PROPIEDAD > ADMIN > DIRECCIÓN > MANAGER > USER). 2FA obligatorio por email en login.

JWT tokens con refresh, sesiones seguras. Datos sensibles (salarios, DNI, IBAN) excluidos de las herramientas IA. Soft deletes: nada se borra realmente, todo es recuperable. 1,104 tests automatizados verifican que todo funciona tras cada cambio.

Comunicación entre departamentos

Sistema de Solicitudes unificado: Diseño, Films, Logística, Catálogo — todo en un buzón. Notificaciones in-app y por email cuando algo cambia. Dashboard Ejecutivo consolida datos de TODOS los departamentos en una vista.
IA (Clau) responde preguntas cruzando datos de cualquier módulo según tu rol. Star Contigo: atención al cliente omnicanal (voz, WhatsApp, web) conectada al mismo backend.
